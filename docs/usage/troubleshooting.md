

## 常见问题

### 1. GLIBC 版本问题
```bash
# 错误信息
./auto-upnp: /lib/x86_64-linux-gnu/libc.so.6: version `GLIBC_2.34' not found

# 解决方案：使用静态编译
make build-static
```

### 2. 无法发现 UPnP 设备
- 确保路由器支持 UPnP 功能
- 检查路由器 UPnP 设置是否启用
- 确认防火墙允许 UPnP 通信
- 检查网络连接状态

### 3. 端口映射失败
- 检查路由器 UPnP 设置
- 确认端口未被其他服务占用
- 查看日志获取详细错误信息
- 检查端口范围配置

### 4. 服务无法启动
- 检查配置文件格式是否正确
- 确认端口范围设置合理
- 查看系统权限和网络访问权限
- 检查日志文件获取错误信息

### 5. Web 界面无法访问
- 确认管理服务已启用
- 检查防火墙设置
- 验证访问地址和端口
- 确认认证信息正确
